# Wifi-Diagnostics-Tool
<!DOCTYPE html>
<html>
Diagnostic tool showing key Wi-Fi metrics: 
<ul>
  <li><b>Signal quality<b>Visual indicator (Excellent/Good/Fair/Poor) based on ping</li>
  <li><b>Ping/latency</b>How responsive your connection is</li>
  <li><b>Download Speed (Mbps)</b></li>
  <li><b>Upload Speed</b>Estimated based on connection info</li>
  <li><b>Connection Type</b>Shows network type (4G, WiFi, etc.)</li>
</ul>
Key features include one click speed test, visual signal strength bars, real-time connection strength monitoring, and a clean UI.
